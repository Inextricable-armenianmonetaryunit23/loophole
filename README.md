## 🧭 What this app does

Loophole helps you test rules, prompts, and legal text by putting them under stress.

You write a set of moral rules in plain language. The app turns them into a formal code. Then it runs two checks:

- **Loophole Finder** looks for cases that follow the rule but break the intent
- **Overreach Finder** looks for cases the rule blocks but should allow

This helps you spot weak wording, narrow exceptions, and broad bans before you rely on the rules.

## ✍️ How to use loophole

Start with short, clear rules. The app works best when your input is specific.

### Example input
- No personal data may be shared without consent
- Emergency access is allowed if it protects life
- Public safety checks may happen in shared spaces

Then review the two result types:

- **Loopholes**
  - hidden gaps
  - edge cases
  - rule misuse
  - technical compliance with bad intent

- **Overreach**
  - false bans
  - blocked help
  - emergency exceptions
  - safe actions marked as unsafe

Use the results to rewrite your rules until they match your intent.

## 🔍 Best ways to write rules

Use short sentences. One idea per line helps the app read your text well.

Good rule writing:

- say what is allowed
- say what is not allowed
- name exceptions
- define key terms
- use examples when a rule is unclear

Try to avoid:

- long nested sentences
- vague words like “reasonable” without context
- undefined terms
- broad claims with no exception
- rules that depend on hidden assumptions

## 🧪 Example workflow

1. Write a policy
2. Run the loophole check
3. Review the cases that expose gaps
4. Run the overreach check
5. Review the cases that block safe actions
6. Update the policy
7. Run the checks again

Repeat this until the rule set matches the result you want.

## 🧰 Troubleshooting

### The app does not open
- Check that the download finished
- Extract the ZIP file if needed
- Try running the app again
- Right-click and choose **Run as administrator** if Windows blocks it

### The app opens, but nothing happens
- Wait for the first check to finish
- Make sure your internet connection is active
- Try shorter input text
- Close other heavy apps and run it again

### The text looks broken
- Increase the window size
- Use the default font settings
- Paste plain text instead of formatted text
- Remove extra line breaks if the input looks messy

### The results feel too broad
- Make the rules more specific
- Add clear exceptions
- Define key words
- Split one large policy into smaller parts
